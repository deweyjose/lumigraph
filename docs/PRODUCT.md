# Product

## Purpose

Lumigraph helps astrophotographers publish outcomes, manage integration data, and explore relevant astronomy context.

## Primary users

- Astrophotographers publishing final images
- Learners consuming integration data and techniques

## Phase model

- Phase 1: posts + integration datasets + downloads
- Phase 2: Astro Hub and stronger publishing ergonomics
- Phase 3: optional lightweight checklists/todos tied to owned content
- Phase 4: AI assistance around posts and integration data

## Current priorities

- Reliable upload/export/download flows
- Clear ownership/visibility controls
- Low-friction authoring and publishing UX

## Non-goals (current)

- Auto-publishing user content
- Overly broad generic file hosting behavior
